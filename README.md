# greenhand-java
菜鸟学java
# 项目说明

本项目为 Java 新手入门项目，集成了以下技术：
- MySQL
- MyBatis-Plus
- SLF4J
- 多线程与线程锁

## 目录结构
- controller：控制层
- dao：数据访问层
- service：服务接口层
- service/impl：服务实现层
- config：配置文件
- dto：入参/出参实体类
- entity：数据库实体类

## 快速开始
1. 配置数据库连接（src/main/resources/config/application.properties）
2. 编写业务代码（controller、service、dao、entity、dto）
3. 运行主类（GreenhandJavaApplication.java）
